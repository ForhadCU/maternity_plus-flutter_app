# [Maternity Plus] 🎉

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

A mobile application designed to support expectant mothers and new parents through the journey of pregnancy and early childhood. This app offers personalized tracking, suggestions, and tools to make the pregnancy experience and baby care simpler and more organized.

## Features ✨

- **Pregnancy Period Tracking**: Keep track of the different stages of pregnancy with detailed week-by-week updates.
- **Daily Data Logging**: Log daily symptoms, weight, and other important health metrics.
- **Personalized Suggestions**: Receive tailored health and wellness advice based on the data entered.
- **Baby Diary**: Document the growth and milestones of your baby with a dedicated baby diary.
- **Growth Tracking Questions**: Guided questions to ensure your baby is meeting developmental milestones.
- **Baby Online Shop**: Convenient access to a baby product store via WebView integration.

## Getting Started 🚀

### Prerequisites

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK: Included with Flutter

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    ```

2. Navigate into the project directory:

    ```bash
    cd your-repo-name
    ```

3. Install the dependencies:

    ```bash
    flutter pub get
    ```

4. Run the app on your emulator or device:

    ```bash
    flutter run
    ```

## Usage 🎯

- **Track Pregnancy**: Enter your pregnancy details, and the app will keep you updated on each stage.
- **Log Daily Data**: Monitor your symptoms and weight to keep a record for yourself and your healthcare provider.
- **Baby Diary**: Record key moments of your baby's life and track developmental progress.
- **Shop for Baby Products**: Use the integrated WebView to browse a variety of baby products.
